# Stockly Tutorial V4 Offline-Friendly

Versão educacional do Stockly criada para ambientes com internet bloqueada ou firewall restritivo.

## Principal diferença

O módulo `app` não possui dependências externas. O projeto usa somente Java, Android SDK nativo, SQLiteOpenHelper e SharedPreferences.
